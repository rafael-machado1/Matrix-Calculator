## Objective

- Calculate any square matrix needed, that includes its determinant

- Be as fast as possible (that's why it's written in C)

- Learn

## Running the code

- Be sure you have GCC or any C compiler

- Create a bin folder by using this command:

```Linux
mkdir bin
```

- Clone this repository and run this command

```GCC
gcc src/main.c -o ../bin/MatrixCalculator
```

- Run the binary

```Linux
cd bin && ./MatrixCalculator
```
